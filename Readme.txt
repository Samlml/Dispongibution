Required files:
deep_sea_coral_sponge_data.csv, downloaded from the NOAA Deep Sea Coral and Sponge Map at https://www.ncei.noaa.gov/products/noaa-deep-sea-coral-and-sponge-map. Make sure this is in the same folder as any used notebooks.


data_analysis_notebook.ipynb:
This notebook contains the initial analysis of the dataset, including functions to sort out the sponge data from the coral data, to seperate the sponge data into classes, and to plot the sponge data relative to depth and latitude.

maptools.py:
This file contains a function for use in the mapping.ipynb notebook. Make sure this is in the same folder as the Mapping.ipynb notebook. This notebook maps demosponges and glass sponges.
